# Early Returns 2026 — Project Memory

## What This Is
Early Returns is a political intelligence dashboard focused on competitive 
U.S. House races (2026 cycle). The product aggregates data and facts — 
fundraising, voting records, race structure, media — with light editorial 
curation on top. Analysis is honest and calibrated to what the data 
actually supports. No forced punditry.

This project is simultaneously:
- A content product for political enthusiasts and engaged voters
- A Gregory (gregoryco.ai) proof of concept for AI-assisted content production
- A learning lab for Claude Code applied to marketing workflows

## Content Philosophy
- Dashboard-first: data aggregated, facts assembled, race structure identified
- Aggregator of others' analysis before originating our own
- Transparency about the AI workflow is a feature, not a liability
- Human judgment drives angle, curation, and editorial voice
- Build expertise this cycle; be fully positioned for 2028

## Race Coverage
- Primary focus: competitive U.S. House races
- Secondary: Senate races of interest
- Race structures: safe seat/intraparty, lean seat, true tossup, 
  incumbent in trouble, open seat, recruited challenger
- Not covering state legislative races

## Data Stack
- **FEC API** — fundraising, cash on hand, burn rate (automated)
- **Congress.gov API** — voting records, party unity scores (automated)
- **Cook / Inside Elections / Sabato** — ratings (manual weekly update)
- **Local and national press** — RSS monitoring, manual curation

## Automation Principles
- FEC and Congress.gov refresh on schedule without manual input
- Ratings and media curation stay in Greg's lane
- Goal: automation handles data gathering so Greg's time goes to analysis

## Voice & Style
- Staccato-legato: compressed, rhythmic, direct
- Confident but calibrated — never overclaims
- Transparent about methodology and uncertainty

## What NOT to Do
- Don't force analysis when data doesn't support a strong take
- Don't reproduce what Cook/Inside Elections/Sabato already do
- Don't cover races without a genuine competitive angle
- Keep this file under 200 lines
